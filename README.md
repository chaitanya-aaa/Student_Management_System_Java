# 🎓 Student Management System (JSP + JDBC)

## 📌 Project Overview

The **Student Management System** is a web-based application developed using **JSP (JavaServer Pages) and JDBC**. It allows administrators to manage student records through basic **CRUD operations (Create, Read, Update, Delete)**.

The system provides a simple interface where users can **register, view, update, and delete student details** stored in a MySQL database.

The application is designed as a **learning project to demonstrate database connectivity and web development using core Java technologies**.

---

## 🎯 Problem Statement

To build a web-based application that enables administrators to **efficiently manage student records** by performing operations such as adding, viewing, updating, and deleting student information.

## 🛠️ Project Modules Covered

### 1️⃣ Student Registration
- Add new student records to the database.

### 2️⃣ View Student Records
- Retrieve and display student data stored in MySQL.

### 3️⃣ Update Student Details
- Modify existing student information.

### 4️⃣ Delete Student Records
- Remove student data from the database.

### 5️⃣ Login & Dashboard
- Provides a login interface.
- Redirects users to the main dashboard.

## 🌟 Key Features

### 📋 Student Record Management
Perform **Create, Read, Update, and Delete** operations.

### 🔗 Database Integration

## 📌 Conclusion

The **Student Management System** demonstrates how core Java technologies such as **JSP and JDBC** can be used to build a dynamic web application connected to a relational database. The project provides hands-on experience with database operations, web server deployment, and user interface development.

> Note: For better architecture, it is recommended to use **JSP + Servlets** to separate business logic from the presentation layer.
